# ML-project-with-MLflow

## Workflow du projet ML

1. Configuration des fichiers YAML (config.yaml, schema.yaml, params.yaml)
2. Définition des entités (Entity) pour structurer les données
3. Implémentation des composants (ex: Data Ingestion, Validation)
4. Création des pipelines d'entraînement et de prédiction
5. Exécution du projet via main.py
6. Développement d'une interface utilisateur (app.py) si nécessaire
